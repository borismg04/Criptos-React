Styles Components 

Que son los Styled Components en React:
    En JSX podmos utilizar HTML con expresiones JavaScript,pero tambien es posible crear un elemento HTML 
    con la Syntaxis de un Componente y añadir propiedades CSS.

Ventaja es reutilizar los estilos CSS en los componentes    

Ventaja de crear los propios Hooks
  La mayoria de los casos es mejor crear Hooks propis , una de las razones es para poder 
  reutilizarlos en una funcion

  Aunque,tambien se peude crear un funcion helper,pero existe otra gran ventaja de crear 
  los propios Hooks y es la forma de incorporar State y mantener el valor 
  de una funcion de forma persistente

  El codigo personalizado tendra todas las vetnajas de React con son:
    State
    effects
    Integracion de otros hooks y el performance

Que es una API 
  API = Application Programming Interface

Son funciones, metodos que ofrece una libreria para ser utilizada por otro Software 
como una capa de abstracción

Una API pone adisposicion recursos que estan alojados en otro servidor o Bases de Datos
